# ⚡ Visuels d'Ultimate (bannières plein écran)

Ces images s'affichent EN PLEIN ÉCRAN DE COMBAT quand un joueur active l'ult.
C'est une illustration/banner horizontale qui couvre toute la zone de combat.

## Convention de nommage
Forme de base    → [templateId]_ult.png
Évolution 1      → [templateId]_ult_evo1.png
Évolution 2      → [templateId]_ult_evo2.png
etc.

## Taille recommandée
Format PAYSAGE : 1280×720 ou 1920×400 px
L'image s'étire pour couvrir TOUTE la zone de combat (comme tes Kamehameha).
Fond transparent ou semi-transparent recommandé pour laisser voir le combat en dessous.

## Exemple : Goku (7 formes)
| Fichier              | Forme                   |
|----------------------|-------------------------|
| goku_ult.png         | Goku base               |
| goku_ult_evo1.png    | Super Saiyen            |
| goku_ult_evo2.png    | Super Saiyen 3          |
| goku_ult_evo3.png    | Super Saiyen Divin      |
| goku_ult_evo4.png    | Super Saiyen Blue       |
| goku_ult_evo5.png    | Signe UI                |
| goku_ult_evo6.png    | Ultra Instinct          |
